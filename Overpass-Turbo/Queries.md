Basic query examples
```
// Search for Baseball Stadium's in USA
sport=baseball and leisure=stadium in USA

// Find Restaurant / Cafe's
amenity=restaurant around "1600 Pennsylvania Ave NW, Washington"
amenity=cafe around "1600 Pennsylvania Ave NW, Washington"
amenity=cafe and brand:Costa in UK
amenity=cafe and brand:Costa in "Edinburgh, UK"

// Find Post Office's in Town/City
amenity=post_office in "Bristol, UK"

// Fins Cycle Route 
route=bicycle in UK
route=bicycle and ref=442 in UK

// Find Guest House in Town/City
tourism=guest_house in "Newquay, UK"

// Find Traffic Mirrors
highway=traffic_mirror

// Find Peaks/Hills or Ridges
natural=peak | hill | ridge

// Find Street Art/Mural's
artwork_type=mural
tourism=artwork

// Find Customer Parking
amenity=parking and access=customers

// Find Gas Stations in Country
amenity=fuel in Libya

// Find Car Wash
amenity=car_wash
car_wash=*

// Find Car Repair shop
shop=car_repair
```
